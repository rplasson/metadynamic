metadynamic.collector.Collect(Generic[K, T]) class documentation

Part of metadynamic.collector (View In Hierarchy)

Object collector.

Generic class to be implemented for Collectable objects of type T, indexed with Hashable key of type K. Keeps two dictionaries {K:T} (a pool with all objects, an activ for only active ones
