def set_param(self, key, val, terminate=(True)):

Set a parameter value.

This will automatically update parameters defined by a relation

If an attempt is done to fix the value of a parameter defined by a relation, the request will be ignored *except* if the 'terminate' flag is set to False (do not use this flag manually, this is intended to be used by _param_init; direct use may result in an inconsistent set of parameters)
