metadynamic.inval

defines invalid values for any type.

Provides:

  • InvalidError: error to be raised when an invalid value is found where unexpected
  • Invalid: generic Invalid class
  • isvalid: function testing the validity of an object
  • invalidint: invalid int object. isvalid(isvalidint) returns False; isinstance(invalidint, int) returns True
  • invalidfloat: invalid float object. isvalid(isvalidfloat) returns False; isinstance(invalidfloat, float) returns True
  • invalidstr: invalid str object. isvalid(isvalidstr) returns False; isinstance(invalidstr, str) returns True
