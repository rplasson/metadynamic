metadynamic.mpi

Control the coordination between thread in MPI runs

Provides

  • Cont: switchable "continue" flag
  • MpiGate: MPI barrier that can be opened/closed by any thread, for on-demand requests of synchronization
  • MpiStatus: Interface to general MPI operations.
  • MPI_STATUS: global MpiStatus object
  • MPI_GATE: global MpiGate object
