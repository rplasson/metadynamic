metadynamic.ruleset

All elements for designing a set of rule, i.e. the Core for model design !

Provides:

  • classes for the description of a rule model:
    • Parameters:
    • Descriptor:
    • Rule:
    • Model:
  • generators to be used for building models:
    • Paramrel generators:
      • parmul
      • arrhenius
      • linproc
    • ConstBuilder generators:
      • kinvar
      • kalternate
      • kdualchoice
    • VariantBuilder generators:
      • novariant_gen
      • singlevariant
      • rangevariant
    • ProdBuilder generators:
      • joiner
      • splitter

def novariant_gen():

Generate a VariantBuilder for reactions with no variant (i.e. only one possible outcome).

The returned VariantBuilder will return a single invalid variant value, thus is intended to be used with ProdBuilder and ConstBuilder that do not use variant for computing the reaction property.

ReturnsVariantBuilder function (type: VariantBuilder)
def singlevariant(num):

Generate a VariantBuilder for reactions with a single variant (i.e. only one possible outcome).

The returned VariantBuilder will return a single valid variant value 'num', thus is intended to be used with ProdBuilder and ConstBuilder that do use variant for computing the reaction property, but in cases where only one of all possible variants is expected to be used.

Parametersnumvariant number to return (type: int)
ReturnsVariantBuilder function (type: VariantBuilder)
def rangevariant(reacnum, first_offset=0, last_offset=0):

Generate a VariantBuilder for reactions with as many variants as the length of the reactant number 'reacnum', ranging from 0 to len(reactants[reacnum]).

A first_offset can be added to change the first variant. A last_offset can be added to change the last variant.

Parametersreacnumnumber of the reactant to be used for computing the variant (type: int)
first_offsetoffset to be added to the first variant (type: int)
last_offsetoffset to be added to the last variant (type: int)
ReturnsVariantBuilder function (type: VariantBuilder)
