metadynamic.inputs

Define parameter json files and a generic Readerclass for reading them.

Provides

  • LockedError: Exception raised when attempting to modify a locked Readerclass
  • Castreader: Extend a Caster for dealing with Readerclass, converting them as dictionary
  • Readerclass: dataclass with interface for reading its data from json files
  • RuleParam: Parameters for a reaction rule
  • RulesetParam: Parameters for a reaction ruleset
  • StatParam: Parameters for statistics
  • MapParam: Parameters for map statistics
  • Param: Run parameters
  • DotParam: Parameters for graphviz CRN representation
