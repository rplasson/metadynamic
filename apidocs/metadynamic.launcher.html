metadynamic.launcher

High level interface to direct run launch

Provides

  • launch: function for launching a simulation from a .json parameter file, a .hdf5 result file from a previous run, with eventual additional parameters, and storing the results in a new .hdf5 file.

Launch a metadynamic run, saves the result in a .hdf5 file, and returns a ResulReader object

It can be launched either form a .json Param file or from a previous .hdf5 result file.
