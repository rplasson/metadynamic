metadynamic.logger

Logging facility, taking into account logging from a MPI run, and the possibility to save logs in a hdf5 file simultaneously to normal logging.

Provides

  • Timer: Simple class for tracking passed processing time from a checkpoint
  • Log: High level class for MPI-aware logging, with log save in hdf5.
  • LOGGER: Global Log object
