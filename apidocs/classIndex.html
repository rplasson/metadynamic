metadynamic API Documentation

Class Hierarchy

  • Chemical[ReacDescr]
    • metadynamic.chemical.Reaction - Chemical describing a specific reaction
  • Chemical[str]
    • metadynamic.chemical.Compound - Chemical describing a specific compound
  • Collect[ReacDescr, 'Reaction']
    • metadynamic.chemical.CollectofReaction - Collect class for storing a pool of Reaction
  • Collect[str, 'Compound']
    • metadynamic.chemical.CollectofCompound - Collect class for storing a pool of Compound
  • Exception
    • metadynamic.ends.Finished - General exception raised at run completion
      • metadynamic.ends.Aborted - Raised for shortened, but correct, run completion
        • metadynamic.ends.InitError - Error during initialization
        • metadynamic.ends.Interrupted - Asked to stop
        • metadynamic.ends.OOMError - Out of Memory
        • metadynamic.ends.RuntimeLim - Runtime limit exceeded
      • metadynamic.ends.BadEnding - Raised for faulty run completion
        • metadynamic.ends.DecrZero - Tried to decrement unpopulated species
        • metadynamic.ends.InputError - Raised when problem are encountered when reading input files
          • metadynamic.ends.BadFile - The provided file is badly formed
          • metadynamic.ends.BadJSON - Bad JSON format
          • metadynamic.ends.FileCreationError - The file couldn't be created
          • metadynamic.ends.FileNotFound - The provided file was not found.
          • metadynamic.ends.NotAFolder - The provided foldername is not a folder
        • metadynamic.ends.InternalError - Something went bad in the code
        • metadynamic.ends.NotFound - No reaction could be find
        • metadynamic.ends.RoundError - Rounding problem/negative probability
      • metadynamic.ends.HappyEnding - Raised for normal run completion
        • metadynamic.ends.NoMore - No more reactions can be processed
        • metadynamic.ends.TimesUp - Time is up
    • metadynamic.inputs.LockedError - Exception raised when attempting to modify a locked Readerclass
  • float
    • metadynamic.inval.InvalidFloat - Invalit float class
  • Generic[K, T]
    • metadynamic.collector.Collect - Object collector.
  • Generic[K]
    • metadynamic.chemical.Chemical - Collectable objects with chemical properties (Generic class)
  • int
    • metadynamic.inval.InvalidInt - Invalid int class
  • metadynamic.caster.Caster - generic type-caster generator
    • metadynamic.inputs.Castreader - Extend a Caster for dealing with Readerclass, converting them as dictionary
  • metadynamic.chemical.Memcalc - Function memoizer
  • metadynamic.collector.Collectable - Generic class, to be derived in classes that will be collected in Collect
    • metadynamic.chemical.Chemical - Collectable objects with chemical properties (Generic class)
  • metadynamic.Crn - Chemical Reaction Network class
  • metadynamic.ends.SignalCatcher - Instances of this class can be used to temporarily catch SIGTERM and SIGINT signal interruptions
  • metadynamic.hdf5.ResultWriter - Storage for all the simulation data and results
  • metadynamic.inputs.Readerclass - dataclass with interface for reading its data from json files
    • metadynamic.inputs.DotParam - Parameters for graphviz CRN representation
    • metadynamic.inputs.MapParam - Parameters for map statistics
    • metadynamic.inputs.Param - Run parameters
    • metadynamic.inputs.RuleParam - Parameters for a reaction rule
    • metadynamic.inputs.RulesetParam - Parameters for a reaction ruleset
    • metadynamic.inputs.StatParam - Parameters for statistics
  • metadynamic.inval.Invalid - For defining Invalid classes of any type. An invalid object of type <T> is an instance of a class that inherits from both Invalid and <T>
    • metadynamic.inval.InvalidFloat - Invalit float class
    • metadynamic.inval.InvalidInt - Invalid int class
    • metadynamic.inval.InvalidStr - invalid str class
  • metadynamic.logger.Log - High level class for MPI-aware logging, with log save in hdf5.
  • metadynamic.logger.Timer - Simple class for tracking passed processing time from a checkpoint
  • metadynamic.mpi.Cont - switchable "continue" flag
  • metadynamic.mpi.MpiGate - MPI barrier that can be opened/closed by any thread, for on-demand requests of synchronization
  • metadynamic.mpi.MpiStatus - Interface to general MPI operations.
  • metadynamic.network.Data2dot - generate a graphviz Digraph from dict description of compounds/reaction
  • metadynamic.network.Graphwriter - Interface for building a graphviz Digraph from a CRN
  • metadynamic.network.Scaler - convert values from one scale to another
  • metadynamic.outputs.Output - Access to output files and folders
  • metadynamic.proba.Probalist - object storing all probalistic events
  • metadynamic.ResultReader - Interface for a hdf5 result file
  • metadynamic.ruleset.Descriptor - tool for describing a compound from its name (categories and properties)
  • metadynamic.ruleset.Model - full description of a rule model
  • metadynamic.ruleset.Parameters - Maintain a set of parameters {name:values} that can be linked with each other.
  • metadynamic.ruleset.Rule - Describes a given rule for building a reaction
  • metadynamic.System - Create, run and control a simulation
  • metadynamic.system.RunStatus - Control the status of a simulation (time, step, memory)
  • metadynamic.system.Statistic - Compute, get and save statustics on a simulation run
  • str
    • metadynamic.inval.InvalidStr - invalid str class
  • ValueError
    • metadynamic.inval.InvalidError - error to be raised when an invalid value is found where unexpected
